Tutorial de Docker

Introducción a Docker

Docker es una plataforma de software que permite a los desarrolladores, administradores de sistemas y otros profesionales, crear, implementar y ejecutar aplicaciones distribuidas de manera eficiente y rápida en diferentes entornos.

Cómo Correr un Contenedor Simple

Para correr un contenedor simple usando Docker, sigue estos pasos:

  1. Paso 1: Abre tu terminal o línea de comandos.
  2. Paso 2: Ejecuta el siguiente comando para descargar la imagen de ejemplo "hello-world" desde Docker Hub y correr un contenedor basado en esa imagen:
docker run hello-world

Este comando buscará la imagen "hello-world" localmente. Si no la encuentra, la descargará desde Docker Hub. Una vez descargada, Docker creará un contenedor a partir de esa imagen y ejecutará el comando definido en la imagen.

Cómo Verificar que el Contenedor Está Activo

Después de correr el contenedor, puedes verificar su estado utilizando el siguiente comando:

docker ps -a

Este comando te mostrará una lista de todos los contenedores en ejecución, así como los que se han detenido recientemente. Si el contenedor "hello-world" está activo, verás una entrada para él en la lista, junto con su ID, nombre, estado, puertos, etc.
