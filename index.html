Docker

Docker es una plataforma de software que permite a los desarrolladores, administradores de sistemas y otros profesionales, crear, implementar y ejecutar aplicaciones distribuidas.

GitHub

GitHub es una plataforma de desarrollo colaborativo de software para alojar proyectos utilizando el sistema de control de versiones Git. Los usuarios pueden colaborar en proyectos, realizar seguimiento de los cambios y versiones, y gestionar el código fuente de manera eficiente.

AWS

Amazon Web Services (AWS) es una plataforma de servicios en la nube ofrecida por Amazon. Proporciona una amplia gama de servicios de infraestructura informática, como almacenamiento, redes, bases de datos, análisis, inteligencia artificial, aprendizaje automático, IoT y mucho más, para ayudar a las empresas a escalar y crecer.
